About the Website

FinancePilot is a Website that helps the user in order to manage their finances, the functionality of the website contains of an OLLAMA-based Chatbot, Along with CSV-File Uploading Functionality, Advice on Financial Aspects.

Frameworks Involved

This project is built using various modern frameworks and technologies:

  • HTML for structuring the content.
  • CSS for styling and design.
  • JavaScript for adding interactivity.
  • Node.js for the backend services.
